用户登录接口

作者: qinyixuan

请求URL

/runapi/login POST

请求参数 application/x-www-form-urlencoded

参数名 类型 必须 描述
id string 否 否 string 用户id
userName string 否 是 string 用户姓名
password string 否 是 string 用户密码

返回结果

{
	"id":"string //用户id",
	"userName":"string //用户姓名",
	"password":"string //用户密码",
	"car":{
		"name":"string //名字",
		"price":"string //价格"
	}
}
